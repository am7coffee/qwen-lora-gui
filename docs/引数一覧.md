# Qwen LoRA GUI 引数一覧

本ドキュメントは、Qwen LoRA GUIで使用可能な全引数をタブ・セクション別に整理した一覧です。

## 概要

GUI は6つのメインタブで構成されており、合計105個のパラメーターが設定可能です：

- **1. モデル・出力** (21個)
- **2. 基本学習設定** (13個)  
- **3. オプティマイザー・学習率設定** (11個)
- **4. ネットワーク** (22個)
- **5. パフォーマンス** (18個)
- **6. ログ・監視** (20個)

---

# 1. モデル・出力

## 1.1 基本出力設定

- `--output_dir` - 学習済みモデル出力先
- `--output_name` - 出力ファイル名
- `--resume` - 学習再開用state

## 1.2 モデルパス設定

- `--dit` - DiTチェックポイントパス
- `--vae` - VAEチェックポイントパス  
- `--vae_dtype` - VAEデータ型
- `--text_encoder` - テキストエンコーダーパス

## 1.3 保存スケジュール設定

- `--save_every_n_epochs` - エポック間隔での保存
- `--save_every_n_steps` - ステップ間隔での保存
- `--save_last_n_epochs` - 最新N個のエポックを保持
- `--save_last_n_epochs_state` - 最新N個のエポック状態を保持
- `--save_last_n_steps` - 最新N個のステップを保持
- `--save_last_n_steps_state` - 最新N個のステップ状態を保持
- `--save_state` - 状態を保存
- `--save_state_on_train_end` - 学習終了時に状態保存

## 1.4 メタデータ設定

- `--metadata_title` - メタデータ：タイトル
- `--metadata_author` - メタデータ：作者
- `--metadata_description` - メタデータ：説明
- `--metadata_license` - メタデータ：ライセンス
- `--metadata_tags` - メタデータ：タグ
- `--no_metadata` - メタデータを無効化

---

# 2. 基本学習設定

## 2.1 基本パラメータ

- `--config_file` - 設定ファイルパス
- `--dataset_config` - データセット設定ファイル
- `--max_train_steps` - 最大学習ステップ数
- `--max_train_epochs` - 最大エポック数
- `--seed` - ランダムシード
- `--training_comment` - 学習コメント

## 2.2 データローダー設定

- `--max_data_loader_n_workers` - データローダーワーカー数上限
- `--persistent_data_loader_workers` - 永続的データローダーワーカー

## 2.3 最適化設定

- `--gradient_checkpointing` - 勾配チェックポイント
- `--gradient_accumulation_steps` - 勾配蓄積ステップ数
- `--mixed_precision` - 混合精度学習
- `--blocks_to_swap` - スワップするブロック数
- `--max_grad_norm` - 勾配クリッピング最大ノルム

---

# 3. オプティマイザー・学習率設定

## 3.1 オプティマイザー設定

- `--optimizer_type` - オプティマイザー種別（テンプレート駆動型）

## 3.2 optimizer_args設定

※ オプティマイザー選択に応じて動的に表示される引数群

## 3.3 学習率設定

- `--learning_rate` - 学習率
- `--lr_scheduler` - 学習率スケジューラー
- `--lr_warmup_steps` - ウォームアップステップ数
- `--lr_decay_steps` - 学習率減衰ステップ数
- `--lr_scheduler_num_cycles` - スケジューラーサイクル数
- `--lr_scheduler_power` - スケジューラー指数
- `--lr_scheduler_min_lr_ratio` - 最小学習率比率
- `--lr_scheduler_timescale` - タイムスケール
- `--lr_scheduler_type` - スケジューラー種別

---

# 4. ネットワーク

## 4.1 ネットワーク設定

- `--network_dim` - ネットワーク次元数
- `--network_alpha` - ネットワークアルファ値
- `--network_module` - ネットワークモジュール
- `--network_weights` - ネットワーク重みパス
- `--network_dropout` - ドロップアウト率
- `--dim_from_weights` - 重みから次元数を取得
- `--scale_weight_norms` - 重みノルムをスケール
- `--network_args` - ネットワーク引数
- `--base_weights` - ベース重み
- `--base_weights_multiplier` - ベース重み乗数

## 4.2 タイムステップ設定

- `--timestep_sampling` - タイムステップサンプリング方式
- `--weighting_scheme` - 重み付けスキーム
- `--guidance_scale` - ガイダンススケール
- `--discrete_flow_shift` - 離散フローシフト
- `--sigmoid_scale` - シグモイドスケール
- `--logit_mean` - ロジット平均
- `--logit_std` - ロジット標準偏差
- `--mode_scale` - モードスケール
- `--min_timestep` - 最小タイムステップ
- `--max_timestep` - 最大タイムステップ
- `--preserve_distribution_shape` - 分布形状を保持
- `--show_timesteps` - タイムステップを表示

# 5. パフォーマンス

## 5.1 アテンション最適化

- `--sdpa` - Scaled Dot Product Attention
- `--flash_attn` - Flash Attention
- `--sage_attn` - Sage Attention
- `--xformers` - xFormers
- `--flash3` - Flash Attention 3
- `--split_attn` - Split Attention

## 5.2 精度・メモリ最適化

- `--fp8_base` - FP8ベース精度
- `--fp8_scaled` - FP8スケール精度
- `--fp8_vl` - FP8 Vision-Language
- `--img_in_txt_in_offloading` - 画像・テキストオフロード

## 5.3 コンパイル最適化

- `--dynamo_backend` - Dynamoバックエンド
- `--dynamo_mode` - Dynamoモード
- `--dynamo_fullgraph` - Dynamo フルグラフ
- `--dynamo_dynamic` - Dynamo 動的最適化

## 5.4 分散処理設定

- `--ddp_timeout` - DDP タイムアウト
- `--ddp_gradient_as_bucket_view` - DDP 勾配バケットビュー
- `--ddp_static_graph` - DDP 静的グラフ

# 6. ログ・監視

## 6.1 ログ出力設定

- `--logging_dir` - ログディレクトリ
- `--log_with` - ログ出力先
- `--log_prefix` - ログプレフィックス
- `--log_tracker_name` - ログトラッカー名
- `--wandb_run_name` - W&B実行名
- `--wandb_api_key` - W&B APIキー
- `--log_tracker_config` - ログトラッカー設定
- `--log_config` - ログ設定

## 6.2 サンプリング設定

- `--sample_every_n_steps` - ステップ間隔でのサンプリング
- `--sample_at_first` - 最初にサンプリング
- `--sample_every_n_epochs` - エポック間隔でのサンプリング
- `--sample_prompts` - サンプリングプロンプト

## 6.3 外部連携設定（HuggingFace等）

- `--huggingface_repo_id` - HuggingFace リポジトリID
- `--huggingface_repo_type` - HuggingFace リポジトリ種別
- `--huggingface_path_in_repo` - HuggingFace リポジトリ内パス
- `--huggingface_token` - HuggingFace トークン
- `--huggingface_repo_visibility` - HuggingFace リポジトリ可視性
- `--save_state_to_huggingface` - HuggingFaceに状態保存
- `--resume_from_huggingface` - HuggingFaceから再開
- `--async_upload` - 非同期アップロード

---

## 注記

- 各引数の詳細な説明と設定可能な値については、GUI上のヘルプテキストを参照してください
- オプティマイザー引数（optimizer_args）は選択したオプティマイザーに応じて動的に表示されます
- 一部の引数は相互に依存関係があります。適切な組み合わせで使用してください
